# Task 3. 상대 날짜와 모호한 표현 다루기

## 이번 태스크에서 배우는 것

`다음주 이 시간`, `오전 중`, `후속 회의 전` 같은 표현을 무리하게 확정하지 않고, 확정 가능한 값과 확인이 필요한 값을 분리합니다.

이번에는 다음 2가지를 작성합니다.

- `Date and ambiguity handling`
- `Procedure` 보강

## 목표 체크

- 후속 회의가 일정 후보로 포함된다.
- `다음주 이 시간`은 기준 시각을 사용해 `2026-05-29 14:00`으로 해석된다.
- 고객 피드백 정리는 `후속 회의 전` 준비 일정으로 남는다.
- 재무팀 비용 추정치 공유는 `2026-05-27 오전`까지는 해석하되 정확한 시간은 확인 필요로 남긴다.
- 유진님의 디자인 시안 업데이트는 제외된다.
- `오전 중`, `후속 회의 전` 같은 표현을 임의의 정확한 시각으로 바꾸지 않는다.
